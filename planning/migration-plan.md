# Docusaurus Migration Plan

## Scope

- Migrate current `docs` and `docs-wiki` content from the Antora wiki.
- Create a separate Docusaurus repo under `8Keep`.
- Target GitHub Pages for the first hosted deployment.
- Prioritize a working content migration before full theme polish.

## Milestones

1. Scaffold the new Docusaurus repo and CI/deploy workflow.
2. Add importer scripts for pages, assets, and sidebars.
3. Convert the current Antora content into Docusaurus docs.
4. Apply an initial jMonkeyEngine brand theme.
5. Validate routes, links, and publish a preview build.

## Streams

- Site scaffolding and deploy
- Antora to Markdown conversion
- Sidebar and route generation
- Asset migration
- Theme and branded UI
- Redirect and broken-link cleanup
- Contributor/admin docs migration
