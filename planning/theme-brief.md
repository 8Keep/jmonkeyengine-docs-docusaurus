# Theme Brief

## Direction

- Preserve jMonkeyEngine identity without cloning the Antora UI.
- Keep the first pass clean, fast, and readable for technical docs.
- Use a branded hero, stronger color system, and custom doc chrome before deep component swizzling.

## Brand Tokens

- Primary accent: warm orange/gold
- Secondary accent: deep ocean blue
- Backgrounds: off-white and deep slate
- Emphasis: high-contrast code, strong callouts, restrained gradients

## Theme Targets

- Navbar and footer
- Homepage hero
- Sidebar contrast and hierarchy
- Code blocks and admonitions
- Inline UI tokens for keyboard/menu/button macros
