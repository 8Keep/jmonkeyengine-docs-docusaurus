# Contribute

There are several ways to contribute to jMonkeyEngine and its ecosystem.

## Contribute to the Engine

The engine source is hosted on GitHub:

- [jMonkeyEngine engine repository](https://github.com/jMonkeyEngine/jmonkeyengine)
- [Open issues](https://github.com/jMonkeyEngine/jmonkeyengine/issues)

Fix bugs, improve features, and submit pull requests for review.

## Improve the Documentation

Documentation matters as much as code.

- [Docs site](/docs/documentation)
- [Original wiki repository](https://github.com/jMonkeyEngine/wiki)
- [Contributor guide](/docs/wiki/wiki_contributor)

## Report Bugs

- [Engine issue tracker](https://github.com/jMonkeyEngine/jmonkeyengine/issues/new)
- [Wiki issue tracker](https://github.com/jMonkeyEngine/wiki/issues/new)

If you are not sure whether something is a bug or a misunderstanding, ask on the [Community Hub](https://hub.jmonkeyengine.org) first.

## Support the Project

- [Donate](/donate)
- help other developers on the forum
- publish tutorials and learning resources
- contribute ecosystem libraries and tools
