# Quick Start

jMonkeyEngine is not tied to a single IDE or SDK. You can start with the online initializer, the SDK, or a custom Gradle or Maven setup.

## The Initializer

The quickest way to begin is the online initializer:

- [Open the jMonkeyEngine Initializer](https://start.jmonkeyengine.org)
- Generate a starter project and import it into your editor

## The SDK

The NetBeans-based SDK is still the fastest packaged environment for many users.

- [Download the SDK](https://github.com/jMonkeyEngine/sdk/releases)
- Review [SDK setup and tooling](/docs/sdk)

## DIY Setup

If you prefer your own editor or build setup:

- Start with [Maven setup](/docs/getting-started/maven)
- Review [Requirements](/docs/getting-started/requirements)
- Follow [Build from sources](/docs/getting-started/build_from_sources)

## First Steps

After your project is running:

- Read [Documentation](/docs/documentation)
- Work through [Beginner tutorials](/docs/tutorials/beginner/hello_simpleapplication)
- Ask questions on the [Community Hub](https://hub.jmonkeyengine.org)
