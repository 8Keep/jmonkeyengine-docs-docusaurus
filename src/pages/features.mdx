# Features

jMonkeyEngine is a feature-rich engine for single-player and networked games across desktop and mobile platforms.

## Platforms

- Windows
- Linux
- macOS
- Raspberry Pi 3
- Raspberry Pi 4
- Android

## Graphics

- Physically based rendering
- single-pass lighting
- instancing
- offscreen rendering
- post-processing
- HDR tone mapping
- particles
- texture atlas support
- geometry shaders

Read more in [Special Effects Overview](/docs/core/effect/effects_overview) and [Materials Overview](/docs/core/material/materials_overview).

## Physics

- built-in Bullet integration
- [Minie](https://github.com/stephengold/Minie) for modern Bullet bindings and soft bodies

## Networking

- built-in TCP and UDP networking API
- high-level RMI support
- ecosystem options like [SimEthereal](https://github.com/Simsilica/SimEthereal)

## GUI

- [Lemur](https://github.com/jMonkeyEngine-Contributions/Lemur)
- Nifty GUI
- JavaFX bridge integrations

## Audio

- WAV, MP3, and OGG
- streaming and buffered playback
- global and positional audio
