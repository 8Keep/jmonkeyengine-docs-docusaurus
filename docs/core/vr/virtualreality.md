# Virtual Reality

## Introduction

Virtual reality is well-supported in jMonkeyEngine 3. The VR support is provided by [user contributed virtual reality libraries](../../contributions/vr/topic_contributions_vr.md), for
example [Tamarin OpenXR](../../contributions/contributions.md#tamarin-openxr).

## Legacy

OpenVr support (and other old virtual reality standards) are documented at [Virtual Reality Legacy support](legacyOpenVr.md) but will be removed in a future release.
