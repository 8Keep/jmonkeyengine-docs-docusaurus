# rise_of_mutants_project

Rise of Mutants project by BigBootsTeam.

Project is hosted here: [https://code.google.com/archive/p/rise-of-mutants/](https://code.google.com/archive/p/rise-of-mutants/)

Features:

- World Editor is Blender (export to Ogre scene).

- Entity System.

- Ready Framework.

- Ready Gameplay.

- Model Viewer tool.

- LightBlow Shader is used.
